JavaScript & CSS

Include JavaScript and CSS files in HEAD tag.
jQuery UI is only required when multiple backgrounds are enabled in IE7,8,9.

                
                    <link rel="stylesheet" type="text/css" href="royal_preloader.css" />
                    <script type="text/javascript" src="jquery.js"></script>
                    <script type="text/javascript" src="royal_preloader.min.js"></script>
                    <script type="text/javascript">
                        Royal_Preloader.config({
                            mode:       'text',
                            text:       'MelonHTML5 - Royal Preloader...',
                            background: '#000000'
                        });
                    </script>

HTML Code

Add a class name "royal_preloader" to the BODY tag and a "royal_preloader" placeholder straight after the BODY tag.

                <body class="royal_preloader">
                    <div id="royal_preloader"></div>
                    ...
                    ...
                    ...
                </body>

Configurations

  • modestring
    Default value: number
    Possible values: text, scale_text, number, line, logo, progress
    number: a loader image is shown, the loading progress is displayed in percentage values and loading details are also displayed on each image load.
    text: a pre-defined text will be displayed in the center of the screen, and fades out from left ro right.
    text: a pre-defined text will be displayed in the center of the screen, and zoomed in from left ro right.
    logo: displaying a logo in the screen center, and fades out from bottom to top.
    line: displaying a youtube like loading progress bar on top of the page.
    progress: displaying a horizontal progress bar width a progress indicator in the center of the page 
                            Royal_Preloader.config({
                            mode: "text"
                        });
  • imagesArray(optional)
    Default value: empty.
    Define additional images to be preloaded. This option is generally used to load images that are not directly displayed as <IMG> tags. (For example: background images used in CSS, third party photo gallery plugins...etc)
                            var images = [
                            'http://www.melonhtml5.com/images/wallpaper/1.jpg',
                            'http://www.melonhtml5.com/images/wallpaper/2.jpg',
                            'http://www.melonhtml5.com/images/wallpaper/3.jpg',
                            'http://www.melonhtml5.com/images/metro_gallery.png',
                            'http://www.melonhtml5.com/images/emu_slider.png',
                            'http://www.melonhtml5.com/images/cut_the_rope.png',
                            'http://www.melonhtml5.com/images/tab.png'
                        ];

                        Royal_Preloader.config({
                            images: images
                        });
  • timeoutInt
    Default value: 10 (seconds).
    When timeout is exceeded, no matter images have been all downloaded or not, the loading screen will finish and your website will be shown.
                            Royal_Preloader.config({
                            timeout: 10
                        });
  • backgroundString
    Default value: "#000000".
    The background colour of the loader.
                            Royal_Preloader.config({
                            backgorund: '#000000'
                        });
  • textString
    Default value: loading...
    Set the loading text. (this is only used in "text" and "scale_text" modes)
                            Royal_Preloader.config({
                            text: "MelonHTML5 - Royal Preloader..."
                        });
  • text_colorString
    Default value: #FFFFFF
    Set the text colour. (this is only used in "text", "scale_text", logo" and "number" mode)
                            Royal_Preloader.config({
                            text_colour: "#FFFFFF"
                        });
  • logostring
    Default value: empty string.
    The URL of the logo image. (this is only used in "logo" mode)
                            Royal_Preloader.config({
                            logo: 'http://www.example.com/logo.jpg'
                        });
  • logo_sizearray
    Default value: [80, 80].
    The size of the logo. (this is only used in "logo" mode)
                            Royal_Preloader.config({
                            logo_size: [80, 80]
                        });
  • showProgressBoolean
    Default value: true.
    Show the loading progress. (this is only used in "number", "logo" and "progress" modes)
                            Royal_Preloader.config({
                            showProgress: true
                        });
  • showPercentageBoolean
    Default value: true.
    Display a % symbol in the loading progress.
                            Royal_Preloader.config({
                            showPercentage: true
                        });
  • cookiestring
    Default value: false.
    When cookie is set, the Royal Preloader only shows once. To reset the cookie, you can either remove the cookie option or use a differnt key value.
                            Royal_Preloader.config({
                            cookie: 'put_a_unique_cookie_string_key_her'
                        });
  • onCompleteFunction
    Default value: null.
    Callback function to be executed after loading is completed.
                            Royal_Preloader.config({
                            onComplete: function() {
                                alert('Woo hoo, loading images are complete');
                            }
                        });
